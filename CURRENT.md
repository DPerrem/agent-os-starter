# Current State

> Working memory. What's true *right now*. AI reads this when the task is about current state — priorities, this-week work, recent decisions.
>
> Update this regularly. Weekly minimum. Daily ideal. If you let it go stale, AI will give you advice based on outdated context.

---

## Right Now

A 2-3 sentence summary of what's happening this period. The reader should be able to skim this and know "what world am I in?"

Example: *Coming out of a quiet December, ramping up [entity X] sales for Q1. New hire starting Feb 3. Personal focus on health after Christmas slip-up.*
