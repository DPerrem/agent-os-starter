# Tasks

> Open work items. AI checks this at the start of sessions where you ask "what's outstanding?" or "what should I work on?"
>
> Keep this lightweight. If you're already using a task system (ClickUp, Linear, Things, OmniFocus), use that as your SSOT and keep this file thin (or delete it). If you don't have one, this is fine for the first 3-6 months.
